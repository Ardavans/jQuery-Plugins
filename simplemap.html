Simple Map Example

Simple Map Summary

The jQuery Simple Map plugin is the easiest way to add Google Maps to your web page, and uses version 3 of the Google Maps API.

Please read the simple maps documentation for instructions on how to use the plugin.

Please visit Google to view their terms of service.

Options

classmodifier
Used to apply class names to the elements created by the plugin. You only need to specify a class modifier if you have a name-clash.
addressList
A list of addresses, or search terms (note that the markers may not be visible given their distance from the first address and the zoom level of the map)
zoom
The zoom level.

Additionally, you can change the map by calling the plugin again with different parameters
