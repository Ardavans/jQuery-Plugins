Simple OpenId Example

Simple OpenId Summary

The jQuery Simple OpenId project is a plugin that converts a plain OpenId form into a more appealing and usable sign-in form.

Please read the simple openid documentation for instructions on how to use the plugin.

Options

classmodifier
Used to apply class names to the elements created by the plugin. You only need to specify a class modifier if you have a name-clash.
baseaddress
Specify a root address of open id images.
